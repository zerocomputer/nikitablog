🚀 Деплой на Timeweb

Для автоматического деплоя используй Timeweb App Platform — он забирает код из GitHub и разворачивает сам. Займёт 2 минуты.

1

Зайди в Timeweb Cloud → App Platform

timeweb.cloud/my/apps

2

Нажми "Создать приложение"

Подключи GitHub-аккаунт, выбери репозиторий zerocomputer/nikitablog

3

Выбери фреймворк: Next.js (Static Export)

Build command: pnpm build
Output directory: out

4

Запусти деплой

Timeweb сам соберёт, развернёт и выдаст технический домен с SSL.

5

Подключи свой домен (опционально)

В настройках приложения укажи свой домен → Timeweb выпустит Let's Encrypt SSL.
